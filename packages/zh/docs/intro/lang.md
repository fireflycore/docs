# 支持语言

Firefly 旨在构建一个多语言的微服务生态系统。目前各语言的支持状态如下：

| 语言 | 状态 | 说明 |
| :--- | :--- | :--- |
| **Golang** | ✅ **Stable** | 核心实现已完成，提供完整的微服务模板 `go-layout`。 |
| **Rust** | 🚧 Planning | 计划中，尚未发布。 |
| **Node.js** | 🚧 Planning | 计划中，尚未发布。 |
| **PHP** | 🚧 Planning | 计划中，尚未发布。 |
| **Kotlin** | 🚧 Planning | 计划中，尚未发布。 |
| **Swift** | 🚧 Planning | 计划中，尚未发布。 |
| **Dart** | 🚧 Planning | 计划中，尚未发布。 |
| **Python** | 🚧 Planning | 计划中，尚未发布。 |

我们欢迎社区贡献其他语言的实现！
