# 常用命令

## 检查
- `firefly env`
- `firefly version`

## 项目
- `firefly create`
- `firefly update`

## 运行
- `firefly run dev`
- `firefly run build`

## buf cli
- `firefly proto add store`
- `firefly proto add module`
- `firefly proto remove store`
- `firefly proto remove module`
- `firefly proto list store`
- `firefly proto list module`
- `firefly proto update`
